# Smart Parking System 🚗

## Overview
Smart Parking System is an Android application developed to help users find available parking spaces for cars and bikes in real time. The app reduces traffic congestion and saves time by guiding drivers to available parking slots.

## Features
- User Registration and Login
- Smart Parking Slot Detection
- Car and Bike Parking Support
- Slot Booking System
- Real-time Slot Availability
- User Friendly Interface

## Technology Used
- Android Studio
- Java / Kotlin
- XML (UI Design)
- Firebase / Local Database

## How to Run the Project
1. Install Android Studio
2. Clone the repository
3. Open the project in Android Studio
4. Run the application on emulator or Android device
